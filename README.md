# Microsite Service

## Tech Stack

-   nodejs
-   typescript
-   express
-   mongoose

## Get Started

1. clone repository

```bash
$ git clone git@github.com:jabardigitalservice/j-site-service.git
```

2. change folder & install node

```bash
$ cd j-site-service & npm install
```

3. Copy `.env` from `.env.example`

```bash
$ cp .env.example .env
```
